Willkommen {{userService.getLocalUser().firstName}} {{userService.getLocalUser().lastName}} im System für die Spielplatzkontrolle der Stadt Winterthur

Falls Sie ein durch die Stadtverwaltung Winterthur akkreditierter Kontrolleur sind, erhalten Sie ein Abbonement auf diese Applikation. Sie müssen sich einloggen, um die Funktionen dieser Applikation verwenden zu können. Falls Sie noch über keine Login-Daten verfügen, wenden Sie sich bitte an die Stadtverwaltung. Zum einloggen verwenden Sie bitte den Login-Button in der oberen rechten Ecke.

Sobald Sie eingeloggt sind, gelangen Sie im linken Applikations-Menü zu den Prüfprozessen für die einzelnen Spielgeräte eines Spielplatzes. Bevor Sie auf die Spielgeräte zugreifen können, müssen Sie einen Spielplatz im Menü-Punkt Spielplatz festlegen.

Diese Applikation ist responsiv, sie funktioniert dadurch auf diversen Endgeräten, z.B. sowohl auf einem Desktop-PC, einem Laptop als auch auf einem Smartphone. Die Darstellung passt sich der jeweiligen Bildschirmgrösse an. Des Weiteren ist diese Applikation offline-fähig, d.h. Sie können sie auch ohne Internetverbindung zur Erfassung benutzen. Hierzu müssen Sie im Online-Modus diese Web-App eingangs einmalig im Browser öffnen und im Menü-Punkt Spielplatz den gewünschten Spielplatz auswählen. Danach können Sie das Erfassungsgerät vom Netzwerk trennen, die Applikation funktioniert weiterhin.

Wenn Sie Daten zu den Spielgeräten eines Spielplatzes in dieser Applikation erfassen, werden sie zunächst nur lokal auf dem Erfassungsgerät gespeichert. Sobald alle Daten zu einem Spieplatz erfasst sind, können Sie den finalen Bericht an das städtische Datenzentrum senden. Dies können Sie im Menü-Punkt Bericht ausführen. Wenn Sie sich an dieser Applikation ausloggen, werden alle bis dahin lokal erfassten Daten gelöscht. Wir empfehlen daher, vor einem Logout den Bericht an das Datenzentrum zu senden.

Wenn Sie den Cache des Web-Browsers leeren, steht diese Applikation offline nicht mehr zur Verfügung. Sie müssen diese Web-App dann nochmals im Web-Browser öffnen, damit diese wieder offline verfügbar ist.
